## 作者简介

Athanasios Papoulis教授于1921年出生于希腊，分别从雅典国家技术大学和美国宾夕法尼亚大学获得了电子工程和数学学位。他于1952年开始在纽约布鲁克林工业大学任教，直至1994年退休。Papoulis教授于2002年4月25日在美国长岛亨廷顿去世，享年81岁。
